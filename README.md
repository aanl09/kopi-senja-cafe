# Kopi Senja Cafe

A simple coffee shop landing page built with AI assistance. This project is intentionally small, clean, and easy to deploy on GitHub Pages.

## Features

- Modern responsive coffee shop landing page
- Menu section with price cards
- Daily promo section
- Simple order form preview
- Contact/location section
- No backend required

## Run locally

Open `index.html` directly in a browser, or run:

```bash
python3 -m http.server 8080
```

Then open:

```text
http://localhost:8080
```
