# Application proof text

I built Kopi Senja Cafe, a simple coffee shop landing page, using an AI-assisted development workflow. The application is a small but complete front-end project that can be hosted on GitHub Pages. It includes a hero section, coffee menu cards, daily promo information, a simple order summary form, and a contact/location section.

The goal of the project is to demonstrate that I can use AI to quickly turn a basic product idea into a working application. The workflow started from a plain idea: create a simple coffee shop web app. With AI assistance, the idea was converted into a project structure, HTML layout, CSS visual design, and JavaScript interaction. The app is lightweight, readable, and easy to maintain.

The core logic is simple: users can browse the menu, choose a drink, enter quantity and customer name, then generate a small order summary. This proves basic front-end interactivity without needing a backend or database. The project is committed to GitHub as proof of AI-assisted building and can be deployed as a live demo.
