# Application proof text

I built Kopi Senja Cafe, a simple coffee shop landing page, using an AI-assisted development workflow. The application is a small but complete front-end project that is hosted on GitHub Pages. It includes a hero section, coffee menu cards, daily promo information, a simple order summary form, and a contact/location section.

The goal of the project is to demonstrate that I can use AI to quickly turn a basic product idea into a working application. The workflow started from a plain idea: create a simple coffee shop web app. With AI assistance, the idea was converted into a project structure, HTML layout, CSS visual design, JavaScript interaction, README documentation, Git commit, GitHub repository, and a live demo deployment. The app is lightweight, readable, responsive, and easy to maintain.

The core logic is simple: users can browse the menu, choose a drink, enter quantity and customer name, then generate a small order summary with the total price. This proves basic front-end interactivity without needing a backend or database. The project is committed to GitHub as proof of AI-assisted building and deployed as a live demo. I also use paid AI tools such as ChatGPT Plus for building, testing, and improving small web applications and automation workflows.
